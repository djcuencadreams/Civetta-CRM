Guía de Integración - Etiquetas de Envío para WordPress

Esta guía te mostrará cómo agregar el formulario de etiquetas de envío a tu sitio WordPress. Hemos desarrollado tres opciones diferentes para que elijas la que mejor se adapte a tus necesidades.

Nota: Todas las opciones de integración son seguras y han sido optimizadas para trabajar con WordPress. La versión más reciente incluye mejor soporte móvil y una experiencia de usuario mejorada.
1

Opción 1: Iframe Optimizado para WordPress (RECOMENDADA)

Esta es la opción más fácil y confiable. Inserta el siguiente código HTML en una página o entrada de WordPress.

Ventajas: Instalación sencilla, alta compatibilidad, mejor diseño responsivo y estilo optimizado. 
<iframe 
  src="https://clothing-sales-tracker-jaradanny.replit.app/wordpress-embed" 
  width="100%" 
  height="1200px" 
  style="border:none;overflow:hidden;" 
  scrolling="no"
  frameborder="0">
</iframe>

Pasos para implementar:

  1. En WordPress, crea una nueva página o edita una existente
  2. Haz clic en el botón para agregar un bloque personalizado y selecciona "HTML personalizado"
  3. Pega el código del iframe en el bloque
  4. Publica o actualiza la página
2

Opción 2: Script Inteligente (Avanzado)

Esta opción usa un script JavaScript para cargar el formulario automáticamente. Requiere dos bloques HTML.

Ventajas: Mayor flexibilidad y carga adaptativa del formulario.

Paso 1: Agrega un contenedor para el formulario

<div id="civetta-shipping-form"></div>

Paso 2: Agrega el script de carga

<script src="https://clothing-sales-tracker-jaradanny.replit.app/shipping-form-loader.js"></script>

Pasos para implementar:

  1. En WordPress, crea una nueva página o edita una existente
  2. Añade un primer bloque HTML y pega el código del contenedor
  3. Añade un segundo bloque HTML (abajo del primero) y pega el código del script
  4. Publica o actualiza la página
Solución de problemas: Si esta opción no funciona correctamente, asegúrate de que no tengas bloqueadores de scripts en tu navegador o plugins que puedan interferir con JavaScript en WordPress.
3

Opción 3: Iframe Directo

Esta opción usa un iframe directo a la URL del formulario.

<iframe 
  src="https://clothing-sales-tracker-jaradanny.replit.app/forms/shipping" 
  width="100%" 
  height="1200px" 
  style="border:none;overflow:hidden;" 
  scrolling="no"
  frameborder="0">
</iframe>

Pasos para implementar:

  1. En WordPress, crea una nueva página o edita una existente
  2. Añade un bloque HTML personalizado
  3. Pega el código del iframe en el bloque
  4. Publica o actualiza la página

Solución de problemas comunes

El formulario no se muestra correctamente en móviles

  • Asegúrate de estar usando la versión más reciente del código (la versión actual está optimizada para móviles)
  • Verifica que el tema de WordPress sea responsive y no tenga restricciones de ancho para iframes
  • Ajusta el valor de height en el iframe si partes del formulario no son visibles

Elementos faltantes o problemas de estilo

  • Utiliza la Opción 1 (iframe optimizado para WordPress) que incluye todos los estilos necesarios
  • Asegúrate de que el tema de WordPress no esté sobreescribiendo los estilos del iframe
  • Verifica que no tengas plugins que modifiquen o bloqueen iframes

El script (Opción 2) no funciona

  • Verifica que hayas añadido tanto el div con el ID correcto como el script
  • Asegúrate de que no haya bloqueadores de JavaScript activos
  • Comprueba que el orden sea correcto: primero el div, luego el script
  • Si persisten los problemas, utiliza la Opción 1 o la Opción 3
